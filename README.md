# How to build

 
### Windows
```Install Visual Studio and open Visual Studio in project folder ```

### Linux / *bsd / mac (unix-like systems (haven't tried it on *bsd, mac but should build))
```bash 
mkdir build
cd build
cmake ..
make
```
